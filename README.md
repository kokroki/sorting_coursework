Исходный код программы расположен в файле sort.cpp. Проект предназначен для экспериментального сравнения алгоритмов сортировки.

В программе реализовано тестирование сортировки вставками, сортировки Шелла и стандартной сортировки C++ std::sort. Для каждого алгоритма измеряется время выполнения.

Сортировки тестируются на трёх типах входных данных: случайных, отсортированных и обратно отсортированных массивах.

Эксперименты проводятся для массивов размером 10, 20, 50, 100, 200, 500, 1000, 2000, 5000 и 10000 элементов. Для каждого размера и каждого типа данных генерируется 500 массивов, после чего вычисляется среднее время работы алгоритмов.

Измерение времени осуществляется с использованием std::chrono::high_resolution_clock. Результаты тестирования выводятся в консоль в микросекундах.
